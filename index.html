Using the charts

Switch between one topic and another using the blue buttons at the top left. Click a small chart at left to change the run chart at right. The small charts are comparable (they use the same scales).

The run chart at right changes scale for each health board. It includes a grey dotted line for a target. Shifts are highlighted with black dots.
